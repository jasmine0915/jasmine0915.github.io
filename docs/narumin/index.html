秋田とは？

秋田県は東北地方の県です。
最近人口が100万人を切りました。
ライバルは山形県です。
秋田出身の有名人は柳場敏郎、佐々木希、高橋優、
野原ひろし等です。

お米がとてもおいしく、
生産量は日本有数となっています。
特にあきたこまちという品種が有名です。

十和田湖
十和田湖は噴火でできたカルデラ湖でとても空気が澄んでいます。
十和田湖周辺の奥入瀬はハイキングコースで有名です。

白神山地
白神山地は世界自然遺産に登録されています。

見るに値すると思います。
